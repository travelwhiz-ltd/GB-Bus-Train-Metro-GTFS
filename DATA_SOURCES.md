# Upstream data sources and licence notes

This file documents the principal upstream data sources used to build the curated
GTFS feed.

## How to read this file

- "Source" describes the upstream dataset or provider
- "Coverage" describes the relevant geography or modes
- "Licence / terms" identifies the governing terms
- "Attribution" gives the minimum wording you should preserve where relevant
- "Notes" captures important downstream conditions or caveats

---

Source:
Traveline National Dataset

Coverage:

Licence / terms:

Notes:
Check the source publication page for any exceptions or additional notices.

## 2. Transport for London (TfL) open data

Source:
TfL Unified API and related TfL open data feeds

Coverage:
London public transport services and related data

Licence / terms:
TfL Transport Data Service terms and conditions

Attribution:
Contains information provided by Transport for London

Notes:
TfL data is published under TfL's own terms, which are based on OGL with
specific amendments. Review TfL branding, attribution, and downstream-use
conditions before redistribution or public display.
